## Getting Started

### Prerequisites

- Node.js

### Setup

Clone the project:

```
git clone https://github.com/blakbox23/album
```

### Run project

Install dependencies:

```
npm install
npm run dev
```

### Run tests

To run tests,

```
npm run test
```
